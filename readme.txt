株式会社翔泳社より刊行の書籍『jQuery 標準デザイン講座』で利用する学習用サンプルファイルです。
本書中でサンプルファイルを使った学習があるLESSONでは、使用するファイルの場所をLESSON冒頭に掲載していますので、記載に従ってファイルを開き、ソースコードを書き加えながら学習を進めてください。

※Windows標準の「メモ帳」やMac標準の「テキストエディット」などでは、文字コードや改行コード（本書p.018参照）が反映されずに表示が崩れてしまう場合がありますので、文字コードを判別できるテキストエディタを使ってファイルを開くようにしてください。

※本書刊行後に判明した誤記載の訂正や補足情報などは、下記URLにて公開しています。ご参照ください。
http://www.shoeisha.co.jp/book/detail/9784798136226#errata
